Prerequisite to run the project
1. Visual studio latest version with Framework 10.0
2. PostgreSQL
3. This AutoShop project has been created using ASP.Net entity framework. Target Framework 10.0

Database:
1. Database Script file available in "DDLScript.txt" file
2. Database connection string is in OnConfiguring method of AutoRepairDbContext.cs file. May need to change according to your environment


Scope:
This is just a prototype with minimal business functionalities which includes
   - Customer module
   - Vehicle module
   - WorkOrder module
   - Logging
